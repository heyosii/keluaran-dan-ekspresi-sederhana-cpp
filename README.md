# Keluaran dan Ekspresi Sederhana (C++)

Repository ini berisi kumpulan jawaban TLX TOKI "Program Dasar dengan C++" part 1 tentang keluaran dan ekspresi sederhana. Setiap file .cpp dibuat sebagai contoh mandiri yang bisa dijalankan langsung.

Penjelasan materi yang lebih lengkap, termasuk konsep dan pembahasan langkah demi langkah, tersedia di halaman Notion berikut:

https://heyosii.notion.site/tlx-toki-1

Repository ini merupakan bagian dari proses belajar dan dokumentasi pribadi.
